Boilerplate
===========

Lightweight HTML5 / javascript application boilerplate based around Require.js and Mocha BDD

Includes:
* Require.js
* Mocha test framework
* jQuery 1.9.0
* Underscore 1.4.3
* Dependency loader, providing CDN fallback and preloading
* normalise.css
* Lightweight HTML5 index.html
